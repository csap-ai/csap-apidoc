# 列宽调整功能使用指南

## ✨ 功能简介

现在你可以自由拖动调整"已选字段配置"表格的列宽了！这个功能已经成功添加到：
- **请求参数管理**模态框
- **返回参数管理**模态框

## 📖 如何使用

### 步骤 1：打开模态框
在 API 管理界面，点击任意接口的"请求参数"或"返回参数"按钮，打开对应的管理模态框。

### 步骤 2：找到"已选字段配置"表格
在模态框右侧，你会看到"已选字段配置"区域，这里显示了已添加的字段列表。

### 步骤 3：拖动调整列宽
1. 将鼠标移动到表格表头任意列的**右边缘**
2. 当鼠标光标变成 **↔** 调整大小图标时
3. **按住鼠标左键并向左右拖动**
4. 松开鼠标完成调整

### 视觉提示
- 鼠标悬停在可拖动区域时，会显示**淡蓝色背景**提示
- 每个列边缘都有一个 10px 宽的拖动热区

## 🎯 可调整的列

### 请求参数表格
| 列名 | 默认宽度 |
|------|---------|
| 字段名称 | 200px |
| 类型 | 150px |
| 请求类型 | 150px |
| 备注 | 200px |
| 必传 | 80px |
| 操作 | 160px |

### 返回参数表格
| 列名 | 默认宽度 |
|------|---------|
| 字段名称 | 200px |
| 类型 | 150px |
| 备注 | 200px |
| 必传 | 80px |
| 操作 | 100px |

## 💡 使用场景

### 场景 1：字段名太长显示不全
当字段名称较长时，可以拖宽"字段名称"列，完整显示字段信息。

### 场景 2：需要更多空间填写备注
拖宽"备注"列，获得更大的编辑区域。

### 场景 3：紧凑显示更多内容
缩小不常用的列（如"必传"、"操作"列），让表格更紧凑，显示更多行。

### 场景 4：根据屏幕调整
在不同分辨率的屏幕上，自由调整列宽以适应显示。

## ⚠️ 注意事项

1. **会话级别保存**
   - 列宽调整仅在当前会话中有效
   - 刷新页面或重新打开模态框后，会恢复默认宽度
   
2. **嵌套字段**
   - 支持对包含子字段的嵌套结构进行列宽调整
   - 所有层级的字段使用相同的列宽设置

3. **最佳实践**
   - 建议保持"操作"列宽度足够，避免按钮换行
   - "必传"列通常不需要很宽，保持 80-100px 即可
