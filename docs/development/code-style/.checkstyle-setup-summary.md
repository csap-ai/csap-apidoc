## 📋 配置详情

### Checkstyle版本
- **Maven插件版本**: 3.3.1
- **Checkstyle核心版本**: 10.12.5

### 主要规则
- ✅ 命名规范（类、方法、变量等）
- ✅ 代码格式（缩进4空格、行长度120字符）
- ✅ 导入管理（禁止星号导入）
- ✅ 空格和换行规范
- ✅ JavaDoc要求（public API）
- ✅ 代码质量检查（空块、简化表达式等）

### 严格程度
- **当前配置**: 警告模式（不会阻止构建）
- `failsOnError`: false
- `failOnViolation`: false
- `violationSeverity`: warning

## 🚀 使用方法

### 命令行

```bash
# 检查代码风格
mvn checkstyle:check

# 生成详细报告
mvn checkstyle:checkstyle

# 构建时自动检查
mvn clean install
```

### IDE集成

#### IntelliJ IDEA
1. 安装 "CheckStyle-IDEA" 插件
2. Settings → Tools → Checkstyle
3. 添加配置文件：`checkstyle.xml`

#### VS Code
1. 安装 "Checkstyle for Java" 扩展
2. 配置文件路径指向 `checkstyle.xml`

#### Eclipse
1. 安装 "Checkstyle Plug-in"
2. 导入 `checkstyle.xml`

## 📊 报告位置

运行 `mvn checkstyle:checkstyle` 后：
- HTML报告：`target/site/checkstyle.html`
- XML报告：`target/checkstyle-result.xml`

## 🔧 自定义配置

如需调整规则，编辑 `checkstyle.xml`：

```xml
<!-- 示例：修改行长度限制 -->
<module name="LineLength">
    <property name="max" value="150"/>  <!-- 改为150 -->
</module>

<!-- 示例：允许更多参数 -->
<module name="ParameterNumber">
    <property name="max" value="10"/>  <!-- 改为10 -->
</module>
```
